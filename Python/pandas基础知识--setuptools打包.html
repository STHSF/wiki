python setup.py test

打包

python setup.py bdist_wheel

问题

问题1

使用python setup.py install安装完成之后, 调用包接口, 显示No module named XXX, setup.py文件中的name跟XXX是一致的, 而且也添加了__init__.py文件

原因

刚开始怀疑目录下面缺少__init__.py文件, 但实际上在查看生成的build文件时发现, 原来将setup.py放在了安装包文件目录的内部, 打包完成之后发现./build/lib/目录下并不是以XXX为主目录的包结构, 而是以XXX命名的目录下的文件, 由此可以看出setup.py文件放错了位置, 而是应该放在以XXX为主目录的包结构的外面, 这样, 打包之后才能将以XXX命名的文件全部打包进去.
