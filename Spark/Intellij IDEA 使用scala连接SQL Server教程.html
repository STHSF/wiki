写在前面

在数据处理过程中免不了会对数据库进行操作，本文主要介绍在intellij下使用scala连接sqlServer数据库。之前我在介绍scala 使用JDBC方式访问Mysql时，java是有关于mysql驱动的依赖包的，但是SQLServer好像没有相关的依赖包。需要从外部添加相关的jar包。
